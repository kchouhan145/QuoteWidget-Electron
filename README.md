# Quto Widget

A simple Electron JS desktop widget application.

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm start
```

### Build the app

```bash
npm run build
```

> **Note:** Make sure you have `electron-builder` configured in your `package.json`.

## Keyboard Shortcuts

- **Reload:** `CTRL+R`
- **Close:** `CTRL+W`
